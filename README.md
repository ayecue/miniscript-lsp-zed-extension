# miniscript-lsp-zed-extension

This extension uses the [MiniScript Languageserver](https://github.com/ayecue/miniscript-languageserver). 

## Configuration

Use any of the [configuration settings](https://github.com/ayecue/miniscript-languageserver/tree/main/packages/node#configuration) that miniscript-languageserver supports within your zed config under `miniscript.settings`.
